# FlowPost_django
# FlowPost - это платформа для монетизации авторского контента. Автор может выкладывать кк платные так и бесплатные посты для своей аудитории, а так же создавать свои кастомные тарифы подписок.

## Ключевые возможности
* Регистрация, Авторизация
* Содаздание тарифного плана подписок
* Просмотр профиля
* Создание поста
* Возможность поддержать автора подпиской
## Технологический Стэк
* Python
* Django Framework
* SQLite3
* HTML, CSS, JS
##  HOWTODO
* python -m venv venv - создание виртуального окружения
* venv\Scripts\activate - активация виртуального окружения
* pip install -r requirements.txt - установка зависимостей
* python manage.py makemigrations - создание миграций
* python manage.py migrate - выполнение миграции
* python manage.py createsuperuser - создание супер пользователя для адним панели
* python manage.py runserver - запуск проекта
