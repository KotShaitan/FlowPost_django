# Этап 1. Планирование
### Структура сайта.
* Страницы регистрации, авторизации
* Страница для пользователя
* Страница для другого пользователя
* Страница Для создания поста
* Страница для создания настройки подписок

### Ключевые сущности.
* Пользователь - User
    * login CharFied
    * password CharField
    * isAuthor BooleanField
    * posts List\<Post>
* Посты - Post
    * author Author
    * header TextField
    * text  TextField
    * images List\<Image>
* Подписка - Subscribe
    * Name TextField
    * Price DecimalField
